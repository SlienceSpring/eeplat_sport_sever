如何注册登录eeplat_appshare

eeplat客服  |  使用帮助  |  2014.03.17

1、登录eeplat后，切换为“从appshare安装”选项。

 

2、初次使用，需要输入匹配信息，点击”查询...“获取信息

 

3、输入信息，点击“登录”，进入appshare系统

 

4、进入appshare系统首页，复制ShareID和ShareSecret信息

5、切换为录入界面，录入ShareID和ShareSecret信息，点击”确定“。

6、输入正确信息后，进入应用列表界面，根据自身需要可以选取应用，点击”安装“。

7、安装过程需要等待一定的时间

7、安装成功后，刷新当前页面，可以切换为你安装的应用
